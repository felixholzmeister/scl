## Single Choice List Task (Eckel/Grossman, 2002)

This oTree app allows for implementing the single choice list task to elicit individual level risk preferences -- as proposed by Eckel/Grossman (2002), Evolution and Human Behavior 23(4) -- in different variants and parameterizations by simply altering the documented variables in config.py.
